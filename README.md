# opssat-doom

DOOM in Space.

```
git clone https://github.com/olafurw/opssat-doom.git
cd opssat-doom/
cd src/
./build-32.sh
mkdir output
./bin/opssat-doom -nosound -nomusic -nosfx -nodraw -iwad ../demos/doom.wad -playdemo ../demos/m1-fast
```

### Source Ports

Based on [doomgeneric](https://github.com/ozkl/doomgeneric)
